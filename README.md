### Visão Geral
Este projeto é um sistema de gerenciamento de empréstimos de livros desenvolvido para facilitar o controle e a administração de empréstimos em uma biblioteca. 
A aplicação permite que os usuários registrem, editem e excluam empréstimos de livros, além de oferecer uma interface amigável para gerenciar o acervo de livros disponíveis.

### Funcionalidades
- **Cadastro de Empréstimos**: Permite registrar novos empréstimos de livros, associando-os a usuários específicos.
- **Edição de Empréstimos**: Funcionalidade para editar informações de empréstimos existentes.
- **Exclusão de Empréstimos**: Possibilidade de excluir registros de empréstimos.
- **Listagem de Empréstimos**: Exibe uma lista completa de todos os empréstimos cadastrados, com detalhes sobre o livro e o usuário.
- **Interface Responsiva**: Utiliza Bootstrap para garantir que a interface seja amigável e acessível em diversos dispositivos.
- **Carrossel de Imagens**: Home page com um carrossel de imagens destacando livros ou anúncios importantes.

### Tecnologias Utilizadas
- **ASP.NET Core MVC**: Framework principal para o desenvolvimento da aplicação.
- **Entity Framework Core**: ORM utilizado para a comunicação com o banco de dados.
- **Bootstrap**: Biblioteca de estilos para uma interface responsiva e moderna.
- **SQL Server**: Banco de dados utilizado para armazenar as informações da aplicação.

### Estrutura do Projeto
- **Controllers**: Contém os controladores responsáveis pelo gerenciamento das requisições HTTP.
- **Models**: Define as classes de modelo que representam os dados da aplicação.
- **Views**: Contém as views Razor que definem a interface do usuário.
- **wwwroot**: Pasta para arquivos estáticos como CSS, JavaScript e imagens.
